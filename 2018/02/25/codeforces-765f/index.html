Codeforces 765F Souvenirs

发表于 2018-02-25 | 分类于 解题报告

题目大意

给定一个序列$\{a_n\}$。
有$q$次询问，每次给定$l,r$，询问$$\min_{l\leq i\lt j\leq r}\{|a_i-a_j|\}$$

$2\leq n\leq 10^5,1\leq m\leq3\times10^5$

题目分析

这题的标算是线段树加上一个很妙的时间复杂度分析，这里我们介绍一个莫队（或者说分块）的做法。
设阈值$T$，对于询问长度小于等于$T$的区间，我们用$O(nT)$的时间复杂度预处理答案，单次询问时间复杂度$O(T)$。
对于长度大于$T$的区间，我们将整个序列按照大小$T$分块，这样的区间两个端点肯定不在同一块内。
考虑在莫队的过程中维护当前区间内所有值按顺序形成的链表。
下面的算法要用到一个重要的性质：对于一个双向链表，如果我们删除这样写：

并且按照先删除先加入的顺序重新加入删除的元素，那么加入可以直接写成这样：

因此下面的算法需要通过精巧的设计来满足这个先删除先加入的顺序。
假设我们在处理左端点在区间$[st,en]$的询问。
先建立$en$到$n$所有元素的双向链表。然后再删除$n$到$en+1$的所有元素，接着再加入$en+1$到$n$的所有元素。在这个过程中，我们能够计算出$en+1$到任意$i(i>en)$的答案，记其为$g_i$。
现在我们开始计算询问的答案，将要处理的询问按照右端点递减顺序排序。建立$st$到$n$所有元素的双向链表。接下来不停执行下面的步骤：

  1. 删除$r_{k+1}$到$r_k+1$的所有元素。
  2. 删除$st$到$en-1$的所有元素。
  3. 加入$en-1$到$l_k$的所有元素，这个过程结合$g_{r_k}$我们就可以算出$k$这个询问的答案。
  4. 加入$l_k-1$到$st$的所有元素。

可以发现这个过程始终满足前面提到的顺序。
时间复杂度，当$n\lt m$时取$T=\frac n{\sqrt m}$就可以做到$O(n\sqrt m)$，$n\geq m$时取$T=\sqrt n$是$O(n\sqrt n)$的（避免前面预处理部分时空超限）。
